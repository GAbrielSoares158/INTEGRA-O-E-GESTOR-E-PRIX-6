Como Criar um Endpoint Real

Para criar um endpoint real em seu servidor:

  1. Crie um script PHP, Node.js ou similar em seu servidor
  2. O script deve aceitar requisições POST
  3. Verifique o token de segurança para autenticar as requisições
  4. Processe os dados recebidos conforme necessário
  5. Retorne status HTTP 200 para confirmar o recebimento

Exemplo de endpoint PHP simples:

<?php
// webhook.php
$token = 'SEU_TOKEN_AQUI';
$received_token = $_SERVER['HTTP_X_EGESTOR_TOKEN'] ?? '';

if ($received_token !== $token) {
    http_response_code(403);
    die('Token inválido');
}

$data = file_get_contents('php://input');
$json = json_decode($data, true);

// Processar os dados recebidos
file_put_contents('webhook_log.txt', print_r($json, true), FILE_APPEND);

http_response_code(200);
echo 'Webhook recebido com sucesso';
?>
